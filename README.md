# Gestión de Cuentas Bancarias

Sistema de gestión de cuentas bancarias desarrollado en Java, con manejo de cuentas de ahorro y corriente, transferencias entre cuentas y notificaciones.

## Funcionalidades
- Creación y gestión de cuentas (ahorro y corriente)
- Transferencias entre cuentas
- Notificaciones y envío de correo (simulado)
- Interfaz gráfica para gestión de cuentas

## Tecnologías
- Java
- POO (herencia, interfaces)
- Swing (interfaz gráfica)
